How it works

1

Create your session

Add polls, quizzes, short text prompts, rating questions, and simple kanban boards in the browser.

2

Share the join link

A unique room code and QR code are generated automatically, and students can also type the code manually.

3

Present live

Step through activities, watch responses and anonymous questions arrive live, then export everything as JSON when the session ends.

📊

Live polls

Ask a question with multiple options. Answers update live while you present.

🧠

Quizzes

Add a correct answer and reveal it when ready. Great for checking understanding.

💬

Short text responses

Collect open-ended answers. Students can submit short written responses on their phones.

⭐

Ratings with comments

Ask for 1-5 star feedback, see the live average, and collect optional comments from participants.

🗂️

Simple kanban boards

Let participants add ideas, links, images, GIFs, and videos into presenter-defined columns.

❓

Anonymous questions

Participants can send questions live during the session, and presenters get a lightweight notification.

📱

Easy joining

Students can scan a QR code, open a direct link, or type the room code manually. No app install, no sign-up.

⬇️

Response export

Close the session and download responses as JSON, including ratings, kanban cards, and anonymous questions.

🔓

Open source

Free forever. Self-host your own copy or use the public site. MIT licensed.
